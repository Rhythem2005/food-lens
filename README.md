# FoodLens AI 🍕🔍

FoodLens AI is an AI-powered food discovery and comparison platform that helps users find the best food options across multiple delivery platforms.

Instead of manually checking different food delivery apps, FoodLens collects restaurant data, analyzes pricing, ratings, delivery times, and recommendations, then helps users make better ordering decisions.

> Current Status: 🚧 Actively Under Development

---

## Project Vision

FoodLens aims to become a smart food comparison assistant that answers:

* Which restaurant offers the best value?
* Which platform is cheaper?
* Which option delivers fastest?
* Which restaurant has the highest rating?
* What is the overall best choice?

Future versions will compare restaurants across multiple delivery platforms and provide AI-powered recommendations with explanations.
